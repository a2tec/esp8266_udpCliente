# esp8266_udpCliente
Exemplo UDP cliente para Arduino ESP8266/ESP32

## Objetivo

Exemplo de comunicação com **ESP8266** e **ESP32**

## Dependências
- ESP8266WiFi ou WiFi (dependendo do dispositivo).
- Bibliotecas WiFiUdp.h.
